Outline

Day 1

  • Brief introduction to Bayesian Inference
  • Hierarchical Modeling
    • examples in BUGS/JAGS/STAN
  • Bayesian meta-analysis
  • Introduction to MCMC

Day 2

  • Predictive distribution model checking
    • missing values handling
  • Bayesian Disease Mapping
  • Prior calibration

Many examples in BUGS/STAN/JAGS are provided along with the scripts for the practicals

Pre-requisites

  • There are no formal pre-requisites
  • Analysis will be performed in the R programming language: http://www.r-project.org/
  • Rstudio http://www.rstudio.com/products/rstudio/ is recommended
  • The following R packages will be used in class:
    • RStan
    • rstanarm
    • R2OpenBUGS
    • BRugs
    • rjags
  • Download and install:
    • Stan http://mc-stan.org/
      • make sure to follow all the instruction and setup correctly the toolchain
    • OpenBUGS http://www.openbugs.net/w/FrontPage
    • JAGS http://mcmc-jags.sourceforge.net/

Getting the slides

  • The slides for this course were created with Rmarkdown: http://rmarkdown.rstudio.com/.
  • They are available from https://github.com/berkeley3/BDA.

  • To re-compile the slides:

    • Download the directory containing the lectures from Github
    • In R open the .Rmd file and set the working directory to the lecture directory
    • Click the KnitHTML button on Rstudio or run the following commands:
library(rmarkdown) 
render("main.Rmd")
